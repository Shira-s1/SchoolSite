# School Site (React + Vite)

This is a mock school website built with React, React Router and the Context API. It includes pages for About, Bookstore, After-School Activities, Contact, and a global Shopping Cart.

Quick start (PowerShell):

```powershell
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

Notes:
- This project uses mock data in `/src/data`.
- Styling uses CSS Modules.
- No backend; payment and email sending are simulated.
